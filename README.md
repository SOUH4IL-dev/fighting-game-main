# 🥊 Fighting Game

A lightweight 2D browser-based fighting game built with **HTML, CSS, and JavaScript**, focusing on core game mechanics such as movement, collision detection, and combat interactions.

---

## 📌 Overview

This project demonstrates the fundamentals of game development in the browser, including:

* Real-time player movement
* Sprite-based animations
* Hitbox collision system
* Health management
* Basic physics (gravity & jumping)

It is designed as a learning project and a foundation for more advanced game features.

---

## 🚀 Live Demo

> *You can run the game locally by opening `index.html` in your browser.*

## 🎮 Controls

### Player 1

| Action     | Key     |
| ---------- | ------- |
| Move Left  | `A`     |
| Move Right | `D`     |
| Jump       | `W`     |
| Attack     | `Space` |

### Player 2

| Action     | Key |
| ---------- | --- |
| Move Left  | `←` |
| Move Right | `→` |
| Jump       | `↑` |
| Attack     | `↓` |

---

## 🧠 Core Concepts

* **Game Loop** – Continuous rendering using `requestAnimationFrame`
* **Collision Detection** – Rectangular hitbox system
* **Sprite Animation** – Frame-based character animation
* **State Management** – Handling player states (idle, run, jump, attack)
